Intégrer une IF

Pour intégrer une IF écrite avec Inform dans un fragment, il faut :

  • Copier le contenu du dossier release (réalisé avec un interpréteur web) de votre IF dans le dossier IF
  • Créer un fragment où vous appeler la page play via une iframe (vous ne pouvez pas directement appelez la page play depuis inficio)

Libre à vous de retoucher le style de votre page play.html via le fichier style.css afin que le design s'intègre mieux dans infictio.
