infictio 3.0 - écriture de récits hybrides

infictio est un framework web visant à faciliter l'écriture de récits interactifs permettant de :

  • créer facilement des liens hypertextes en JS
  • profiter de la dimension multimedia du web
  • intégrer des fictions interactives écrites avec Inform
  • tirer profit des effets d'animation de JS et de Jquery

1. Le système hypertexte

infictio permet de créer des liens qui appellent dynamiquement des fragments textuels. Chaque lien appelle un fragment. Un fragment est un fichier html se trouvant dans le dossier view.

Pour créer un lien, il faut utiliser la fonction page().

La fonction page

Le premier argument désigne le lien sur lequel cliquer. Le second, l'élément qui va être modifié dynamiquement. Le troisième donne le chemin relatif vers la vue appelée.

page('.pointer', '#infictio', 'views/part1/frag1.html');

Organisation

Au chargement de l'application, le fichier infictio.js est chargé. Il appelle la vue définie par la variable start (dans notre cas main.html).

Le contenu de main.html vient remplacer le contenu de la div #infictio dans la page index.html. C'est cette dernière qui sera changée dynamiquement en fonction des liens par infictio.

2. Dimension multimedia

Il est très facile d'intégrer des contenus multimédia dans infictio, il suffit de les ajouter directement en tant qu'élément html ou par le biais d'iframe dans une vue html.

Par exemple en cliquant sur ce lien, vous chargez dynamiquement un fragment contenant une iframe.

Il est également possible de manipuler le son en intégrant une librairie comme buzz.js.
